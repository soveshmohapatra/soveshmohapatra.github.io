Deconstructing the NeRF Kernel

Part 3: 10.64 dB From One Trick

By Sovesh Mohapatra • May 2026
Overview

Two MLPs. Same architecture. Same data. Same training budget. The only difference: one of them encodes coordinates through sin/cos at 6 exponentially-spaced frequencies before the MLP sees them. The PSNR difference is 10.64 dB.

Setup

Target. 64×64 RGB image: 8×8 checkerboard, red circle, blue square, yellow diagonal line. Model. 4-layer MLP, hidden 128. ReLU + sigmoid. AdamW lr=5e-4, 2000 iterations.

A $10.64$ dB PSNR improvement corresponds to a $10.6\times$ reduction in MSE. The Fourier-encoded model fits the image more than an order of magnitude better with $8\%$ more parameters and the same training time.

Training Trajectories

The encoded model reaches the no-encoding model's final PSNR ($11.13$ dB) after fewer than 100 iterations — $20\times$ fewer training steps. It plateaus around iteration 1400 at $\sim 22$ dB. The unencoded model is still descending at iteration 2000 but heading toward an asymptote in the low teens.

Visual Reconstruction

No PE: a smooth gradient of muted colors. Checkerboard invisible. Circle and square are fuzzy blobs. Diagonal line has disappeared entirely.

With Fourier features: recognisable. Checkerboard cells visible. Sharp circle and square boundaries. Yellow diagonal streak present.

Why Fourier Features Fix Spectral Bias

The ReLU MLP's spectral bias is provable: its NTK decays at high frequencies, so gradient descent converges faster on low-frequency components. Fourier features change the input representation in a way that flattens the effective kernel across frequencies. After encoding, the input is the same set of features regardless of where in coordinate space we are — the MLP no longer has to manufacture high frequencies; it has to linearly combine them.

From 2D to NeRF

The 3D NeRF pipeline takes the same coordinate-MLP-plus-Fourier-features kernel and adds: cast a ray, sample 3D positions, run each sample through the coordinate MLP to predict color + density, accumulate via the volume-rendering integral. The architecture and the encoding are the same as ours. The geometry is in the rendering, not the model.

The Bigger Lesson

Architecture is not always the answer. The pivotal innovation behind every neural-field method — NeRF, SIREN, Instant-NGP, every Gaussian-splatting precursor — was an input-representation trick. A 4-layer MLP is enough capacity to represent a 3D scene; what was missing was the right way to feed coordinates into it.
